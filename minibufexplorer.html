vim学习

MiniBufExplorer

快捷键

  • Tab : 向前循环切换到每个buffer上
  • Shift - Tab : 向后循环切换到每个buffer上
  • Enter : 打开光标所在的buffer
  • d : 删除光标所在的buffer

命令

在一般模式下

// 打开当前buffer的下一个buffer
:bn

// 打开当强buffer的上一个buffer
:bp

// 打开编号为<num>(即每个buffer前面的数字)的buffer, 
:b<num>
